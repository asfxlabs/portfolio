# Portfolio Website

This is my personal portfolio website built using React, Tailwind CSS, Framer Motion, and Lenis.

The website showcases my projects, skills, development workflows, and creative work through cinematic UI design, smooth animations, and interactive project case studies.

## Tech Stack

- React
- Tailwind CSS
- Framer Motion
- React Router
- Lenis Smooth Scroll

## Features

- Cinematic modern UI
- Smooth scrolling experience
- Interactive project pages
- Animated screenshot showcase
- Responsive design
- Glassmorphism effects

## Run Locally

```bash
npm install
npm run dev
